This program performs a salary analysis system for employees using C.It starts by collecting student details like name, registration number, and section.The user (HR) enters salaries of up to 10 employees.
 A menu-driven system lets the user choose different analysis options.
 Option 1 displays all entered salaries.
 Option 2 calculates and displays the total and average salaries. Option 3 shows the highest and lowest salary.
 Option 4 counts how many employees earn above or below the average.
